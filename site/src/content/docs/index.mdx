A hands-on C#/.NET recap for an experienced **Node.js + TypeScript** developer. It doesn't try to cover everything — it covers the **fundamental differences** between the two stacks, because the other ~70% you already know. Every topic has a **lesson** — *where the real code lives*: concepts mapped to your TS knowledge, and the build itself, typed in as you read — and **exercises with solutions**: validation drills that prove, break, and stress what you just built. Topics 5–10 build one **Payment API** end to end (register, login, transfer, balance) plus the Node payment processor it calls.
