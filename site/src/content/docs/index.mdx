A hands-on C#/.NET recap for an experienced **Node.js + TypeScript** developer. It doesn't try to cover everything — it covers the **fundamental differences** between the two stacks, because the other ~70% you already know. Every topic has a **lesson** (concepts, always mapped to your TS knowledge), **exercises** (hands-on, building one Loan Application API across the whole course), and **solutions** (full working code + interview talking points).
